# Immagini di sfondo login

Metti qui i 3 file richiesti dall'effetto sfondo nella pagina di login, con questi nomi esatti:

- `login-bg-1.jpg` — appare non appena nel campo Email si scrivono le prime 2 lettere "ro"
- `login-bg-2.jpg` — appare quando il campo Email inizia con "robert"
- `login-bg-3.jpg` — appare quando il campo Email corrisponde esattamente a "robertbaciu967@gmail.com"

Puoi usare `.jpg`, `.png` o `.webp`: basta rinominare il file con lo stesso nome (cambiando l'estensione anche in `style.css` se non usi `.jpg`).
