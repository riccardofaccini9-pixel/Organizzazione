# Immagini di sfondo login

- `loginBackground.jpg` — sfondo standard della pagina di login, sempre visibile finché non scatta uno degli effetti sotto.

Effetto progressivo in base a cosa viene digitato nel campo Email:

- `login-bg-1.jpg` — appare non appena si scrivono le prime 2 lettere "ro"
- `login-bg-2.jpg` — appare quando il campo inizia con "robert"
- `login-bg-3.jpg` — appare quando il campo corrisponde esattamente a "robertbaciu967@gmail.com"

Puoi usare `.jpg`, `.png` o `.webp`: basta rinominare il file con lo stesso nome (cambiando l'estensione anche in `style.css` se non usi `.jpg`).
